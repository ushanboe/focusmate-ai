# FocusMate AI - Web App

Local LLM-powered task breakdown timer that works in the browser.

## Status

- ✅ Works on iPhone Safari
- ✅ Works on desktop browsers (Chrome/Edge)
- ❌ Cache API issues on Android browsers (see native-app for solution)

## Deployment

This is the web PWA version deployed to Vercel.

For the native Android app, see: https://github.com/ushanboe/focusmate-android

## Tech Stack

- React 19
- TypeScript
- Vite
- Web-LLM (@mlc-ai/web-llm)
- Capacitor (for native builds)

## Features

- Task breakdown using local LLM
- Step timers with completion tracking
- Favorites system
- Responsive design
